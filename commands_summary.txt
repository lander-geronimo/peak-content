1. Clean raw data -> python3 -m src.etl.clean_tiktok --input data/raw/tiktok_merged_data_deduplicated.csv --output data/processed/posts.parquet
   - Standardizes schema, normalizes timestamps, computes hours_live, cleans hashtags, and logs missing values.

2. Build features & trend metrics -> python3 -m src.features.build_features --input data/processed/posts.parquet --output data/features/training_set.parquet --trend-summary reports/trend_metrics.json
   - Constructs temporal/content/engagement features, trend spike signals, and creates the training label `is_viral`.

3. Train models -> python3 -m src.models.train --features data/features/training_set.parquet --model-dir models --report reports/model_metrics.json
   - Runs Stratified 5-fold CV across Logistic Regression, Random Forest, and Gradient Boosting, logging metrics and saving the best pipeline.

4. Evaluate -> python3 -m src.models.evaluate --features data/features/training_set.parquet --metrics reports/model_metrics.json --report reports/model_eval.md
   - Uses a hold-out split to report accuracy/F1/ROC-AUC and, with --generate-figures, saves confusion matrix & ROC curve.

5. Predict (batch) -> python3 -m src.models.predict --model models/random_forest.joblib --features data/features/training_set.parquet --output reports/predictions.csv --print
   - Scores every row, writes `viral_probability` & predictions; prints metrics if ground truth labels exist. Supports --video-id, --single-json, and --prompt for interactive testing.
